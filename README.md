| `[PHONE_PLACEHOLDER]` | Your WhatsApp number (e.g., 919876543210) | All pages — WhatsApp button & footer |

| `ratnatrayainstitute` (sitemap/robots) | Your actual GitHub username (if different) | sitemap.xml, robots.txt |

## 🚀 Deployment — GitHub Pages

1. Create a GitHub repository named `ratnatrayainstitute.github.io`
2. Upload all files maintaining the folder structure above
3. Go to **Settings → Pages**
4. Source: `main` branch, folder: `/ (root)`
5. Save — your site goes live in 1-2 minutes at `https://ratnatrayainstitute.github.io/`
